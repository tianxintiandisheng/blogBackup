# 使用github搭建个人网站 



1. 在github上搭建网站 即username.github.io（username是你的github用户名）
2. 把自己的域名（如zhangsan.com）绑定到username.github.io上

具体操作：[如何用github搭建自己的个人网站](https://blog.csdn.net/u012075670/article/details/54581015 )
