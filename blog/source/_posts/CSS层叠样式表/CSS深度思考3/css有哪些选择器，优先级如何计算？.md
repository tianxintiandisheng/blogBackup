# 选择器

类别选择器、标签选择器、id选择器、通用选择器、后代选择器、子选择器、伪类选择器、群组选择器、相邻同胞选择器  

# 优先级  

1. 在属性后面使用！important会覆盖页面内任何位置定义的元素样式。 

2. 作为style属性写在元素内的样式 

3. id选择器  

4. 类选择器  

5. 标签选择器  

6. 通配符选择器 

7. 浏览器自定义或子元素集成父类的样式
